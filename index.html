isoverse
open-source data tools for
stable isotopes
The isoverse is a suite of platform-independent, open-source data tools designed for the efficient and reproducible processing of stable isotope data across disciplines.
Check out the available packages and code. If you like the functionality that isoverse provides, please let us know about your experience and ideas and help spread the word.
